The tools

The tools you'll learn to connect

You don't need to master all of them. You need to understand how they fit together — and when to use which.

Data sources — where information lives
Google Sheets, Airtable, forms, CRMs, databases. You'll learn to identify data sources and understand their structure before connecting or moving anything.
Automation platforms — how data moves
Make.com and Zapier are the bridges between systems. You'll learn to configure triggers, map data fields, apply logic, and route information without code.
Dashboards — where data surfaces
Looker Studio, Metabase, and native platform dashboards. You'll learn to connect data sources and build views that make information visible and actionable for your team.
Internal tools — where teams interact with data
Retool, Glide, and Softr let you build interfaces on top of your data. You'll learn to configure these so your team can interact with data without touching the underlying system.

Common questions

FAQ

Everything you need to know before getting started.

Do I need any technical background?
None at all. The platform is designed for people who work with data every day but don't think of themselves as technical. If you've used a spreadsheet, you have enough background to start. We begin with the concepts before any tools.
Will I actually learn to connect tools — or just read about them?
Both — in the right order. We start by building understanding (the mental model for how data flows), then move into practical, hands-on configuration with real tools. Every learning unit ends with something you've actually done, not just read about.
What if I can't find the training I need in the library?
Submit a custom how-to request. Tell us the specific tool, workflow, or data question you need help with — we'll scope it, quote a fee upfront, and deliver a tailored how-to guide or walkthrough built around your exact situation. Completed how-to's are added to the library for all subscribers.
How are custom how-to requests priced?
Fees are based on the complexity of what's involved — a simple "how do I connect X to Y" starts from $39, while a multi-step walkthrough covering several tools might be $149 or more. You'll always receive a clear quote before any work begins, so there are no surprises.
Is this about building software, or learning about data?
Learning about data and the connections between systems — not building software. The focus is on literacy, fluency, and practical skill with the tools that bridge data and technology. You'll be able to configure, connect, and interpret — not code or develop from scratch.
What tools will I need?
Most tools in the curriculum have free tiers that are sufficient for learning — Google Sheets, Make.com, Airtable, and Looker Studio all have free plans. We'll guide you on when (if ever) a paid plan makes sense for your situation.
